# Big Data Modeling Project

## Overview
This project demonstrates a big data modeling pipeline, from raw data ingestion to analysis-ready datasets, using Apache Spark and relational databases.

## Key Features
- **Data Sources:** Simulated customer, product, and transaction data.
- **Technologies Used:**
  - Apache Spark for data processing and modeling.
  - Parquet for efficient storage.
  - PostgreSQL/MySQL for relational database analysis.
- **Schema:** Star schema with fact and dimension tables.

## Project Structure
- **data/**: Raw and processed data.
- **scripts/**: Python scripts for ETL, modeling, and database loading.
- **sql/**: SQL scripts for schema creation and analysis queries.
